== Опис ==

Повнофункціональний онлайн-читач Біблії для WordPress — аналог azbyka.ru/biblia + ekzeget.ru.

== Функціонал ==

**Читання тексту:**
* 70+ книг Біблії (Старий та Новий Завіт, включаючи неканонічні)
* 21+ переклад (рус, укр, англ, грец, євр, лат, нім, фр, ісп, пол, кит, яп та ін.)
* Паралельний переклад у 2 колонки (як azbyka: ?Ex.1&r~utfcs)
* Виділення слів Христа кольором (червоний/зелений/синій)
* Перехресні посилання між віршами
* Номери розділів з швидкою навігацією

**Толкування (як ekzeget.ru):**
* Толкування Святих Отців до кожного вірша
* Адмін-панель для додавання толкувань
* Бічна панель з миттєвим доступом до толкувань

**Налаштування відображення (як azbyka):**
* 5 тем: білий, бежевий, сірий, темний, сепія
* Вибір шрифту (засічки, без засічок, моноширинний, Vollkorn)
* Розмір тексту від 12 до 32px
* Ширина колонки: вузька/середня/широка/адаптивна
* Жирний текст, вирівнювання по ширині, переноси
* Пряма мова курсивом
* Сховати/показати номери віршів
* Таймер нижньої панелі (0.6с — завжди)
* Збереження позиції при виході
* Копіювання з/без номерів, з переносами, з лапками

**Пошук:**
* Повнотекстовий пошук по всій Біблії
* Фільтр за книгою
* Пагінація результатів
* Підсвітка знайденого тексту

**Закладки (для авторизованих):**
* Збереження закладок з кольором та заміткою
* Перегляд та навігація з бічної панелі

**Звіти про помилки:**
* Виділити текст → повідомити про помилку
* Адмін-панель для перегляду звітів

**Адаптивний дизайн:**
* Повна адаптація для мобільних (320px+)
* Складена бічна панель на мобільних
* Touch-friendly керування
* Клавіатурна навігація (←→ між главами)

**Адмін-панель:**
* Dashboard зі статистикою
* Імпорт текстів (вірш за рядком)
* Управління перекладами

== Встановлення ==

1. Завантажте ZIP та розпакуйте в /wp-content/plugins/
2. Активуйте плагін
3. Перейдіть в Bible Reader → Імпорт текстів
4. Імпортуйте тексти Біблії
5. Сторінка /bible/ створена автоматично
6. Або використовуйте шорткод [bible_reader]

== URL формат ==
* /bible/#Mt.1&r — Від Матвія 1, Синодальний
* /bible/#Gen.1&k — Буття 1, Огієнко
* Паралельний: /bible/#Ex.1&r~utfcs
